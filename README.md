# vihang/homebrew-tap

Homebrew formulas for personal/public projects.

## Available formulas

| Formula | What | Project |
|---------|------|---------|
| `yaamux` | Spawn N AI coding agents in parallel git worktrees inside a tiled tmux grid | https://github.com/vihang/yaamux |

## Usage

```bash
brew tap vihang/tap
brew install yaamux       # installs `yaamux` + `ymx` (3-char alias)
brew upgrade yaamux
```

Formulas here are bumped (manually or via the source repo's release workflow)
when the source project tags a new version.
